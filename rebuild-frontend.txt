## API Endpoints (бэкенд)

### Auth
- POST /api/auth/login - вход
- POST /api/auth/register - регистрация
- POST /api/auth/telegram/initiate - инициация Telegram auth
- POST /api/auth/telegram/verify-code - проверка кода

### Character
- GET /api/character/me - получить персонажа текущего пользователя
- POST /api/character - создать персонажа
- GET /api/character/:id - получить персонажа по ID

## Общие стили (index.css)

### Градиентный светящийся текст
Добавлены универсальные CSS классы для красивого оформления текста:

- `.text-gradient-gold` - базовый золотой градиент со свечением
- `.text-gradient-gold-intense` - интенсивное свечение (для заголовков h1)
- `.text-gradient-gold-soft` - легкое свечение (для небольшого текста)
- `.text-gradient-red` - красный градиент (для опасных элементов)
- `.text-gradient-silver` - серебряный градиент (для нейтральных элементов)

**Использование:**
```tsx
<h1 className="text-gradient-gold-intense">Login</h1>
<span className="text-gradient-gold">Beautiful Text</span>
```
