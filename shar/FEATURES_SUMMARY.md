# Краткий список реализованных фич

## BACKEND

### Авторизация
- JWT авторизация + ручки для аутентификации (login, refresh, logout)

### Модули
- Авторизация (генерация токенов, валидация)
- Персонажи (создание, прокачка, распределение стат-поинтов, выносливость)
- Бои (real-time через WebSocket, раундовая система)
- Инвентарь (экипировка, заточка предметов, продажа)
- Подземелья (с монстрами разной сложности)
- Предметы (оружие, броня)
- Лут (генерация после боя)

### Механика
- Классы персонажей (Warrior, Mage, Rogue)
- Система характеристик и прокачки
- Выносливость (ограничение боев)
- Заточка предметов (до +3, каждый уровень +10%)
- Экипировка (5 слотов: оружие, шлем, броня, перчатки, ботинки)
- Зоновая боевая система (голова, тело, ноги)
- AI монстров
- Генерация лута

### База данных
- Prisma ORM + PostgreSQL

## FRONTEND

### Стор
- Redux Toolkit + RTK Query для состояния и API

### Авторизация
- Автоматический вход при загрузке
- ProtectedRoute для защиты страниц
- Автоматический refresh токена

### Роутинг
- 6 защищенных страниц + 1 публичная (создание персонажа)

### Страницы
- Создание персонажа
- Дашборд (инфо о персонаже)
- Подземелья
- Кузница (заточка)
- Инвентарь
- Прокачка
- Бои (PvE через WebSocket)

### Боевая система
- Выбор зон атаки/защиты
- Визуализация результатов боя цветными бордерами (зеленый - попадание, красный - промах/урон, синий - блок)

### UI
- Интерфейс без эмодзи
- Отображение характеристик и предметов

## МЕХАНИКА

### Прогрессия
- Опыт и уровни
- Стат-поинты за уровень
- Заточка экипировки

### Экономика
- Золото за победы и продажу
- Стоимость заточки

### Боевая система
- Инициатива
- 3 зоны атаки/защиты
- Расчет урона и защиты
- Генерация лута

### Выносливость
- Восстановление каждые 5 минут
- Ограничение боев

## ТЕХНОЛОГИИ

### Backend
- NestJS, Prisma ORM, PostgreSQL, Socket.io, Passport.js, JWT, class-validator

### Frontend
- React, TypeScript, Redux Toolkit, RTK Query, React Router, Socket.io-client, Vite
